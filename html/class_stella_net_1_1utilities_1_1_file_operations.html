Reads a fits formatted spectrum file and outputs a spectrum.Spectrum object.

Parameters
file_paththe path to the file that is to be read
table_numthe table number of the .fits file containing the wavelength, flux, and error values
wave_headerthe string header of the wavelength values (if wavelength obtained via CDELT1, CRVAL1, and NAXIS1 use '0')
flux_headerthe string header of the flux values (use '0' if flux vals are all that is in data)
error_headerthe string header of the error values (may be '0' to use zeroes)
parse_paramstrue if the application should parse parameters from the filename
read_rangethe range of wavelengths to include in nm and the number of points (i.e. [400, 600]), can be None if you want to read all. First value is inclusive, second value is exclusive. If the exact points do not exist then their closest match will be read (default None).
Returns
a StellaNet spectrum.Spectrum object generated from the file that was read
