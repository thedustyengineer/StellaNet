Object representing a stellar spectrum (or synthetic spectrum) and all of its associated parameters.

Parameters
selfthe object instance
wavelengthsthe spectrum wavelength values in Angstroms
fluxesthe normalized spectrum flux values
errorsthe per-wavelength errors in the flux (normalized accordingly)
vsini_applied(default: False) bool indicating if the spectrum has been rotationally broadened
vsini_value(default: 0) int indicating the vsini in km/s of the spectrum
noise_applied(default: False) bool indicating if the spectrum has been perturbed with an SNR value
noise_value(default: 0) int indicating the SNR
radial_velocity_shift_applied(default: False) bool indicating if the spectrum has been red or blue shifted
radial_velocity_shift(default: 0) a float indicating the radial velocity shift applied to the spectrum in km/s
is_synthetic(default: True) bool indicating if the spectrum is synthetic or not
snrThe desired signal to noise ratio
